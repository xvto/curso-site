# Curso de HTML5 e CSS3 - Curso em Vídeo

Repositório criado para armazenar exercícios, desafios e projetos desenvolvidos durante o curso de HTML5 e CSS3 do professor Gustavo Guanabara, no canal Curso em Vídeo.

## 📚 Sobre o curso

O curso aborda os fundamentos do desenvolvimento web utilizando:

- HTML5
- CSS3
- Responsividade
- Estruturação de páginas
- Semântica
- Estilização moderna

Curso realizado através da plataforma e canal oficial do Curso em Vídeo.

## 🚀 Objetivo

Praticar desenvolvimento front-end, organizar os estudos e acompanhar minha evolução na criação de sites.
